# Telegram Bot for Secret Santa

## Описание проекта

Телеграм-бот для организации "Тайного Санты" — праздничного розыгрыша подарков. Бот позволяет:
- Регистрировать участников
- Собирать информацию об имени и пожеланиях каждого
- Администратору проводить случайный розыгрыш для определения дарителей
- Уведомлять пользователей о том, кому они дарят подарок

## Требования

**Python:** `>=3.13`

## Зависимости

- `aiogram>=3.22.0` — асинхронный фреймворк для работы с Telegram Bot API
- `redis>=7.1.0` — клиент Redis для работы с хранилищем состояний

## Переменные окружения (`.env`)

Обязательные переменные:

| Переменная   | Описание |
| ----------   | -------- |
| `BOT_TOKEN`  | Токен Telegram бота |
| `ADMIN_ID`   | ID администратора (целое число) |
| `REDIS_PATH` | Путь к папке для хранения aof файлов |
| `DB_PATH`    | Путь к папке для хранения sqlite файла |
